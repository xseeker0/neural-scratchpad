# neural-scratchpad

**neural-scratchpad** is a personal laboratory and documentation archive for exploring the depths of Computer Science, with a primary focus on Artificial Intelligence.

The goal of this repository is not just to utilize libraries, but to understand the **underlying mechanics** of algorithms by implementing them from scratch and dissecting their behaviors.

## Project Roadmap

- **Section 1: PyTorch Fundamentals** – Core concepts, data plotting, and dataset loading.
- **Section 2: Basic Neural Networks** – Introduction to Multi-Layer Perceptrons (MLP) and fundamental architectures.
- **Section 3: Common Deep Learning Architectures**
    - **3.1 Convolutional Neural Networks (CNN)** – Image processing and spatial feature extraction.
    - **3.2 Non-Sequential Models** – Advanced architectures like ResNet (Residual connections).
    - **3.3 Attention Mechanisms & Transformers** – The foundation of modern sequence modeling.
    - **3.4 Transfer Learning & Fine-tuning** – Leveraging pre-trained models for downstream tasks.
- **Section 4: Computer Vision (CV)**
    - **4.1 Classification** – Single and multi-label image categorization. (Coming soon...)
    - **4.2 Attention on Image (ViT)** – Implementing Vision Transformers and patches. 
    - **4.3 Object Detection** – Identifying and locating multiple objects. (Coming soon...)
    - **4.4 Segmentation** – Semantic and instance segmentation. (Coming soon...)
- **Section 5: Natural Language Processing (NLP)** – Basic and advanced tasks in language processing. (Coming soon...)
- **Section 6: Generative AI** – Exploring the world of Generative AI. (Coming soon...)
